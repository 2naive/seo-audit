Сайт traumeel.ru имеет хорошую базовую техническую конфигурацию (HTTPS, редиректы, корректный robots.txt), однако страдает от ряда критических SEO-проблем. Главная страница полностью лишена тега H1. Meta description на главной и нескольких внутренних страницах существенно превышает рекомендуемый лимит (276 символов против нормы 70–160). Сервер не использует сжатие (gzip/brotli), отсутствуют заголовки безопасности, кэширование фактически отключено (max-age=3). Внутренние страницы имеют критически нарушенную структуру H1 (5–9 заголовков H1 вместо одного).

🔴 Критические ошибки

1. Отсутствует H1 на главной странице

Важность: Высокая Сложность: Низкая

На главной странице traumeel.ru нет ни одного тега <h1>. Все заголовки в слайдере оформлены как H2. Поисковики используют H1 как главный сигнал о теме страницы.

<h1>Траумель® С — противовоспалительный препарат при травмах</h1>

2. Meta description значительно превышает допустимую длину

Важность: Высокая Сложность: Низкая

Главная: 276 символов (норма 70–160). /moms/: ~220 символов. Длинные описания обрезаются в выдаче и снижают CTR.

Траумель® С — препарат для лечения воспалений, травм и ушибов. Гель, мазь, таблетки и капли. Подходит детям с 3 лет. (~138 симв.)

3. Отсутствует gzip/brotli сжатие

Важность: Высокая Сложность: Низкая

Сервер не передаёт заголовок Content-Encoding. Страницы передаются без сжатия, что увеличивает объём в 3–7 раз и замедляет загрузку.

gzip on; gzip_comp_level 6; gzip_types text/html text/css application/javascript application/json;

4. Отсутствуют HTTP-заголовки безопасности

Важность: Высокая Сложность: Низкая

Не установлены: Strict-Transport-Security, X-Frame-Options. Критично для YMYL-тематики (фармацевтика/здоровье).

add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always; add_header X-Frame-Options SAMEORIGIN always; add_header X-Content-Type-Options nosniff always;

5. Множественные H1 на внутренних страницах

Важность: Высокая Сложность: Средняя

/sport/: 5 тегов H1. /moms/: 9 тегов H1. Это нарушает семантику и размывает ключевое слово для поисковиков.

/* Оставить один H1 с ключевым словом, остальные заменить на H2 */ <h1>Траумель® С для спортсменов</h1> <h2>Применение</h2>

6. Кэширование фактически отключено (max-age=3)

Важность: Высокая Сложность: Низкая

Cache-Control: max-age=3, must-revalidate — браузер перезапрашивает ресурсы каждые 3 секунды. Статика должна кэшироваться на 1 год.

# HTML: max-age=3600, статика: max-age=31536000, immutable

7. 19 изображений с пустым alt=""

Важность: Высокая Сложность: Средняя

Продуктовые изображения (таблетки, капли, мазь) имеют пустой alt. Google и Яндекс используют alt для понимания контекста и ранжирования в поиске по картинкам.

<img src="traumeel-pills.png" alt="Таблетки Траумель® С — 50 шт">

8. Sitemap содержит только 8 URL

Важность: Высокая Сложность: Средняя

Для фармацевтического сайта 8 страниц критически мало. Страница /accept-cookies/ включена без необходимости. Нет страниц препаратов, статей.

Добавить все значимые страницы. Убрать /accept-cookies/ (добавить noindex). Настроить автообновление через Yoast SEO.
