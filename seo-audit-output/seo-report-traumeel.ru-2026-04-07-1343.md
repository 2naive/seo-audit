## 🔴 Критические ошибки

### 1. Отсутствует H1 на главной странице
**Важность: Высокая** | **Сложность исправления: Низкая**

На главной странице traumeel.ru нет ни одного тега `<h1>`. Все содержимое слайдера оформлено как H2. Поисковые системы (Яндекс и Google) используют H1 как главный сигнал о теме страницы. Отсутствие H1 ведёт к снижению релевантности страницы по ключевым запросам.

**Пример исправления:**
```html
<h1>Траумель® С — противовоспалительный препарат при травмах и воспалениях</h1>
```

---

### 2. Meta description значительно превышает допустимую длину
**Важность: Высокая** | **Сложность исправления: Низкая**

- Главная: 276 символов (норма 70–160)
- /moms/: ~220 символов
- /sport/: ~160 символов (на границе)

Длинные описания обрезаются в поисковой выдаче, что ухудшает CTR и внешний вид сниппета.

**Пример исправления (главная):**
```
Траумель® С — гомеопатический препарат для лечения воспалений, травм и ушибов. Гель, мазь, таблетки и капли. Подходит детям с 3 лет.
```

### 3. Отсутствует gzip/brotli сжатие
**Важность: Высокая** | **Сложность исправления: Низкая**

Сервер не передаёт заголовок `Content-Encoding`. HTML-страница передаётся без сжатия, что увеличивает время загрузки в 3–7 раз. Скорость страницы — прямой фактор ранжирования Яндекса и Google.

**Пример исправления (nginx.conf):**
```nginx
gzip on;
gzip_comp_level 6;
gzip_types text/html text/css application/javascript application/json image/svg+xml;
gzip_min_length 1000;
```

---

### 4. Отсутствуют HTTP-заголовки безопасности
**Важность: Высокая** | **Сложность исправления: Низкая**

Не установлены: `Strict-Transport-Security`, `X-Frame-Options`, `Content-Security-Policy`. Это снижает оценку безопасности сайта в инструментах Lighthouse и ухудшает доверие поисковиков в YMYL-тематике (здоровье/фармацевтика).

**Пример исправления (nginx.conf):**
```nginx
add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
add_header X-Frame-Options SAMEORIGIN always;
add_header X-Content-Type-Options nosniff always;
```

### 5. Множественные H1 на внутренних страницах
**Важность: Высокая** | **Сложность исправления: Средняя**

- /sport/: **5 тегов H1**
- /moms/: **9 тегов H1**

Каждый из этих заголовков претендует быть "главной темой" страницы, что дезориентирует поисковиков и размывает ключевое слово.

**Пример исправления:**
```html
<!-- Оставить один H1 -->
<h1>Траумель® С для спортсменов — лечение спортивных травм</h1>
<!-- Остальные заменить на H2 или H3 -->
<h2>Применение</h2>
<h2>Выберите удобный Траумель® С</h2>
```

### 6. Кэширование фактически отключено
**Важность: Высокая** | **Сложность исправления: Низкая**

`Cache-Control: max-age=3, must-revalidate` — браузер будет перезапрашивать HTML каждые 3 секунды. Это создаёт избыточную нагрузку на сервер и замедляет повторные посещения.

**Пример исправления:**
```nginx
# Для HTML:
add_header Cache-Control "public, max-age=3600, must-revalidate";
# Для статики (CSS/JS/изображения):
location ~* \.(css|js|jpg|png|gif|svg|woff2)$ {
    add_header Cache-Control "public, max-age=31536000, immutable";
}
```

### 7. 19 изображений с пустым alt=""
**Важность: Высокая** | **Сложность исправления: Средняя**

На главной странице 19 изображений имеют пустой атрибут `alt=""`. Пустой alt означает, что изображение декоративное — но среди них продуктовые фотографии (traumeel-pills-new-x2.png, traumeel-c-drops-new-x2.png и др.), которые должны иметь описательный alt.

**Пример исправления:**
```html
<img src="traumeel-pills-new-x2.png" alt="Таблетки Траумель® С — 50 шт">
<img src="traumeel-c-drops-new-x2.png" alt="Капли Траумель® С — флакон 30 мл">
```

### 8. Sitemap содержит только 8 URL
**Важность: Высокая** | **Сложность исправления: Средняя**

Для фармацевтического/медицинского сайта 8 страниц в sitemap — критически мало. Страница `/accept-cookies/` включена в sitemap без необходимости. Нет страниц препаратов, статей, информационных разделов.

**Пример исправления:**

- Добавить все страницы о препаратах, составе, противопоказаниях, применении
- Настроить автоматическое обновление через Yoast SEO

## 🟡 Предупреждения

### 9. TTFB = 0.60 секунды
**Важность: Средняя** | **Сложность исправления: Средняя**

Time To First Byte выше рекомендуемого значения 0.3с. Возможные причины: медленные PHP-запросы, отсутствие page cache, долгие запросы к базе данных WordPress.

**Рекомендация:** Установить плагин page cache (WP Rocket, W3 Total Cache, LiteSpeed Cache) и проверить медленные SQL-запросы.
